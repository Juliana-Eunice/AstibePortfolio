Activity #2 – Operators

Part I — Guided Programs
Problem 1 — Bitwise Operators

Create a Java program using bitwise operators that will produce the output: true, 10, false, 15.

BitwiseOperators.java
package act2operators; public class BitwiseOperators { public static void main(String[] args) { int x = 10; // 1010 int y = 5; // 0101 int z = 15; // 1111 boolean result1 = x > y; // true int result2 = x & z; // 1010 & 1111 = 1010 = 10 boolean result3 = x == y; // false int result4 = x | y; // 1010 | 0101 = 1111 = 15 System.out.println(result1 + ", " + result2 + ", " + result3 + ", " + result4); } }

DifferentOperators.java
package act2operators; public class DifferentOperators { public static void main(String[] args) { int num1 = 8; int num2 = 4; boolean check = true; System.out.println((num1 > num2) && check); // boolean #1 System.out.println(num1 << 1); // numeric #1 int unaryCase = num1++ + ++num1; System.out.println(unaryCase); // numeric #2 System.out.println(unaryCase % num2); // numeric #3 num2 *= 2; System.out.println(num2); // numeric #4 System.out.println(num2 == num1); // boolean #2 } }

Activity #3 – Simple Wallet
A console-based wallet program with a menu loop for viewing balance, adding money, and spending money — with input validation and a transaction counter.
SimpleWalletAstibe.java
package simplewallet; import java.util.Scanner; public class SimpleWalletAstibe { // class name public static void main(String[] args) { // main method Scanner input = new Scanner(System.in); // Scanner double bal = 3000; // set balance double addBal = 0, spendBal = 0; int choice = 0; int transCTR = 0; // my counter variable while (choice != 4) { // while loop // Menu System.out.println("\n===== WALLET MENU ====="); System.out.println("1. View Balance"); System.out.println("2. Add Money"); System.out.println("3. Spend Money"); System.out.println("4. Exit\n"); System.out.println("Transaction " + (transCTR+1) + ":"); System.out.print("Choice: "); choice = input.nextInt(); // switch-case switch(choice) { case 1: System.out.printf("Output: Balance = %.2f%n", bal); transCTR++; break; case 2: System.out.print("Add: "); addBal = input.nextDouble(); bal += addBal; System.out.printf("Output: New Balance = %.2f%n", bal); transCTR++; break; case 3: System.out.print("Spend: "); spendBal = input.nextDouble(); System.out.print("Output: "); // if-else if statement if (spendBal > 1000) { System.out.println("Exceeds maximum spend (1,000)."); } else if (spendBal > bal) { System.out.println("Exceeds balance."); } else { bal -= spendBal; System.out.printf("New Balance = %.2f%n", bal); transCTR++; } break; case 4: System.out.println("\nTransaction Count: " + transCTR); break; default: System.out.println("Invalid choice, choose only 1-4."); } } } }

Activity #4 — Grade Evaluation System
Computes the average of three grades (prelim, midterm, finals) and uses an if-else chain feeding into a switch statement to print the corresponding remark.
GradeEvaluationAstibe.java
package gradeevalsystem; public class GradeEvaluationAstibe { public static void main(String[] args) { double prelim = 85; double midterm = 78; double finals = 90; double average; int range = 0; average = (prelim + midterm + finals) / 3; System.out.printf("Prelim Grade: %.2f", prelim); System.out.printf("\nMidterm Grade: %.2f", midterm); System.out.printf("\nFinal Grade: %.2f", finals); System.out.printf("\n\nAverage: %.2f\n", average); if (average >= 90) { range = 1; } else if (average >= 80) { range = 2; } else if (average >= 75) { range = 3; } else { range = 4; } switch(range) { case 1: System.out.println("Remark: Excellent"); break; case 2: System.out.println("Remark: Good"); break; case 3: System.out.println("Remark: Fair"); break; case 4: System.out.println("Remark: Failed"); break; default: } } }
Console — GradeEvaluationAstibe
Prelim Grade: 85.00 Midterm Grade: 78.00 Final Grade: 90.00 Average: 84.33 Remark: Good
Learnings
Control Structures (If-Else, Switch) Arithmetic Operations Print Formatting (printf)

This was simple to make. I didn’t have to create a switch-case but I decided to do so because I thought it would help me practice control structures more. After hardcoding the grades and calculating the average with arithmetic operators, I used an if-else structure to map those averages to specific range values. Those values then fed into a switch to classify the final performance (like "Excellent" or "Good") making it easier to follow. I also utilized the printf statement more for the formatting!

Activity #5 – Personal Expense Tracker
A multi-method expense tracker that takes a budget and four expense categories as input, then computes the total, checks against the budget, and displays a personalized savings message if the user spent under half their budget.
ExpenseTrackerAstibe.java
package act5; import java.util.Scanner; public class ExpenseTrackerAstibe { // DISPLAY PROGRAM TITLE public static void title() { System.out.println("====================================="); System.out.println(" EXPENSE TRACKER by ASTIBE"); System.out.println("====================================="); } // CALCULATE TOTAL EXPENSES public static float calculateTotEx(float a, float b, float c, float d) { return a + b + c + d; } // CHECK TOTAL EXPENSES public static String checkExceed(float total, float budget) { String message; if (total > budget) { message = ">> Budget exceeded! Spend more wisely."; } else { message = ">> Expenses are within your set budget."; } return message; } // DISPLAY TOTAL EXPENSES AND BUDGET STATUS public static void displayStatus(float a, float b, String message) { float leftOver = b - a; // remaining budget System.out.println("\n======== SUMMARY OF EXPENSES ========"); System.out.printf("Total expenses: %.2f%n", a); System.out.printf("Budget status: You have %.2f left.%n", leftOver); System.out.println(message); } // PERSONALIZED MESSAGE (spent less than or half of budget) public static void displayAccom(float tot, float budget) { if (tot <= budget/2 && budget != 0) { System.out.println("\nGood job! You saved a ton, continue living humbly."); } } // MY MAIN METHOD public static void main(String[] args) { Scanner input = new Scanner(System.in); // for user input title(); // display title System.out.print("Input your budget: "); float budget = input.nextFloat(); System.out.println("Input your expenses:"); System.out.print("- Food: "); float food = input.nextFloat(); System.out.print("- Transportation: "); float transpo = input.nextFloat(); System.out.print("- Shopping: "); // another expense category float shopping = input.nextFloat(); System.out.print("- Other expenses: "); float others = input.nextFloat(); float totalExp = calculateTotEx(food, transpo, shopping, others); String mess = checkExceed(totalExp, budget); displayStatus(totalExp, budget, mess); displayAccom(totalExp, budget); input.close(); } }
